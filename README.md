# Progam ini digunakan untuk menghitung total biaya pengiriman paket berdasarkan
-Berat paket (kg) -Jarak pengiriman (km) -volume paket (panajng x lebar x tinggi)
Biaya pengiriman ditentukan oleh: -Jika jarak ≤ 10 km, maka biaya per kilogram = Rp 4.250 -Jika jarak > 10 km, maka biaya per kilogram = Rp 6.000 -Jika volume > 100 cm³, maka ditambah biaya Rp 50.000
